# Dart-Practice-Concept

widget catalog in flutter : https://docs.flutter.dev/ui/widgets
Mutable (পরিবর্তনযোগ্য) List / Map / Set 
Immutable (অপরিবর্তনযোগ্য)  final , const , static const

 
StatelessWidget immutable: StatelessWidget নিজে change রাখে না কিন্তু change দেখাতে পার
StatelessWidget এমন widget যার ভেতরে নিজস্ব state (data) থাকে না যেটা বদলালে widget নিজে rebuild হবেে


Responsive = Screen size  একই UI থাকে, কিন্তু layout পরিবর্তন হয়  | স্ক্রিন সাইজ অনুযায়ী UI adjust করা |  Mobile vs Tablet

Adaptive = Platform / Device পুরো UI style বদলাতে পারে | Material (Android) vs Cupertino (iOS) | Android vs iOS
